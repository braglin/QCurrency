# :moneybag: QCurrency :moneybag:

Vb.net lightweight desktop criptocoins currency monitor widget

ScreenShot:<br>
![Win7](https://github.com/braglin/QCurrency/raw/master/pic.png)<br>


## How to use
1- Run the QCurrency.exe application (QCurrency/bin/Debug)

2- Choose the cryptocoin that you want to monitor and a widget will appear

3- Click&Hold to move the widget to the desired position
